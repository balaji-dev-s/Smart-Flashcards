Smart Flashcards - Study Smarter, Not Harder.

Smart Flashcards is a completely dynamic, interactive flashcard application that I developed using HTML, CSS, Bootstrap, and vanilla JavaScript. I created it with one goal in mind: to make studying easier and more effective by using spaced repetition, a proven technique that helps learners remember information for a much longer time.

Users can create as many decks as they want, organize their cards, and move everything around using smooth drag-and-drop interactions. They can study through a user-friendly flashcard view, access a library of ready-to-use decks, and export their data as JSON for easy backup.

This project shows my ability to build a complete, user-friendly learning tool from the ground up, and it also highlights my strengths in UI design, UX thinking, front-end development, and problem-solving.

What I Learned From Building Smart Flashcards

Building Smart Flashcards helped me grow quickly as a frontend developer. I also strengthened my UI and UX skills by designing clean layouts, smooth navigation, and meaningful empty states that feel intuitive. Along the way, I learned how to structure code better with modular JavaScript and reusable components, while solving real issues like edge cases, responsiveness, and data persistence.
