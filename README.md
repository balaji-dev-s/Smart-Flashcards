# Smart Flashcards – Study Smarter, Not Harder

Smart Flashcards is a fully dynamic, interactive flashcard web application built using **HTML, CSS, Bootstrap, and Vanilla JavaScript**.  
The goal of this project is simple: **make studying more effective and less overwhelming** by applying the proven learning technique of **spaced repetition**.

This application allows users to create, manage, and study flashcards in an intuitive and distraction-free environment while keeping all data safely stored in the browser.

---

## Key Features

### 1. Create, Edit & Delete Decks
- Create unlimited flashcard decks
- Edit decks anytime
- Delete decks when no longer needed

### 2. Drag & Drop Reordering
- Reorder cards and decks using smooth drag-and-drop interactions
- Improves organization and study flow

### 3. Export Decks (JSON)
- Export decks as JSON files
- Useful for backup, sharing, or restoring progress

### 4. Pre-Built Library
- Ready-to-use decks included
- Add library decks to your collection with one click

### 5. Difficulty Marking (Easy / Medium / Hard)
- Each card can be marked by difficulty
- Improves spaced repetition accuracy

### 6. Progress Tracking
Users can track:
- Cards studied
- Accuracy rate
- Upcoming review sessions
- Learned vs unlearned cards

### 7. Smooth & Clean UI
- Responsive design
- Animated flashcards
- Clean layouts and transitions
- Meaningful empty states for better UX

##  UI / UX Philosophy
